Machine Learning

"At its core Machine learning is teaching a machine, any computer, to react on its own instead of being fed commands. Machine Learning is the first step to creating Intelligent devices, devices that can think for themselves, make decisions without the need for human supervision. The goal for unsupervised learning is to create a model that defines the hidden distribution in the data in order to learn more about the data. There is no concept of right or wrong here. Algorithms, or machines, come up with what they feel defines the data set in the most optimum way." Eager to read more?
